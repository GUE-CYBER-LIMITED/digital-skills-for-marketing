## What is a Website?

A **website** is like a **digital book** stored on the internet that you can read, interact with, or shop from using your Windows computer or Android phone. It‚Äôs made up of webpages (like chapters) that you access through a web browser, such as Microsoft Edge or Chrome.

Think of a website as a store in a giant online mall:

- The **web server** is the store‚Äôs building, holding all the products (webpages).
- The **web browser** is your car, driving you to the store via a road (URL).
- The **webpage content** (text, images, buttons) is the products on display, built with HTML, CSS, and JavaScript.

Websites let you do amazing things, like watch videos on YouTube, shop on Amazon, or chat on social media!

### Why Are Websites Important?

Websites are the backbone of the internet because they:

- Share information (e.g., news, tutorials, or recipes).
- Connect people (e.g., social media or forums).
- Power businesses (e.g., online stores or streaming platforms).

## Key Concepts of How Websites Work

To understand websites, let‚Äôs break down the essential pieces that make them tick, with examples for your Windows computer, Android phone, or any device.

### 1. Web Server

- **What it is**: A **web server** is a powerful computer that stores websites and sends them to your device when you visit them. It‚Äôs always online, ready to serve webpages.
- **Example**: When you visit `www.google.com` on your Windows computer, a Google web server sends the search page to your browser. On your Android phone, the same happens when you open the YouTube app or website.
- **Illustration**: Imagine a web server as a librarian in a massive library (the internet). When you ask for a book (webpage), the librarian fetches it for you instantly.

### 2. HTML (HyperText Markup Language)

- **What it is**: **HTML** is the skeleton of a webpage, defining its structure, like the text, headings, and images.
- **Example**: On a news website, HTML creates the title, paragraphs, and picture placeholders.
- **Illustration**: Think of HTML as the blueprint of a house.

### 3. CSS (Cascading Style Sheets)

- **What it is**: **CSS** is the stylist that makes a webpage look pretty, adding colors, fonts, and layouts to the HTML structure.
- **Example**: On Amazon, CSS makes the buttons blue, text bold, and the layout organized.
- **Illustration**: Picture CSS as the interior designer of the house.

### 4. JavaScript

- **What it is**: **JavaScript** adds interactivity and dynamic features to a webpage.
- **Example**: On Netflix, JavaScript plays a video when you tap the "Play" button.
- **Illustration**: Imagine JavaScript as the electrician who adds smart lights and sensors.

## Web Hosting

**Web hosting** is like renting space in the internet‚Äôs mall for your website. It‚Äôs a service that stores your website on a web server and keeps it online 24/7.

### How it works:

1. A person or company creates a website using HTML, CSS, and JavaScript.
2. They pay a hosting provider to store it on a web server.
3. The server connects to the internet, making the website accessible via a URL.

**Illustration**: Web hosting is like renting a shop in a busy market. The shop (web server) holds your goods (webpages), and customers (users) visit via the market‚Äôs roads (internet).

---

## How Websites Work: The Big Picture

1. **You Enter a URL**: Type `www.bbc.com` in Microsoft Edge or tap a bookmark in Chrome.
2. **Your Browser Requests the Page**: The browser sends a message to the web server.
3. **The Web Server Responds**: The server sends back the HTML, CSS, and JavaScript files.
4. **Your Browser Builds the Page**: The browser displays the content using the received files.

**Illustration**: Visiting a website is like ordering a custom pizza:
- **You Order** (Enter a URL)
- **Request Sent** (Order goes to kitchen/server)
- **Server Responds** (Chef makes pizza with ingredients)
- **You Enjoy** (See and use the webpage)

### Activity 2: Create a Simple Webpage

**Goal**: Build a basic webpage using HTML.

#### Steps (Windows):

1. Open **Notepad**.
2. Paste this code:

```html
<!DOCTYPE html>
<html>
<head>
    <title>My First Webpage</title>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>This is my first webpage, created on Windows!</p>
</body>
</html>
```

3. Save the file as `mywebpage.html` - Choose **"All Files"** as the file type.
4. Double-click the file to open it in your browser.

### Write down:
- **What you saw**
- **How the OS and browser helped**
